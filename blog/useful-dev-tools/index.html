Why do I care?#

Having a consistent workflow for development reduces friction when switching to a new machine, hence increasing developer velocity.

How to do it?#

Install Zsh#

Zsh is an extended version of the Bourne Shell (sh) like Bash, but it offers more easily configurable options especially when paired with Oh My Zsh.

brew install zsh

  • Check which shell: echo $0
  • Switch to zsh: chsh -s $(which zsh)
  • Switch to bash: chsh -s $(which bash)

Install Oh My Zsh#

Oh My Zsh is an open source, community-driven framework for managing your zsh configuration.

sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

Read more about the install.sh file from ohmyzsh repo if interested. Check if there is any error and check to see if ~/.oh-my-zsh exists.

Change the theme by modifying ZSH_THEME=”robbyrussell” in ~/.zshrc. Then, run exec $SHELL to pick up the change.

Install Powerlevel10k#

Powerlevel10k is not a built-in theme of Oh My Zsh, so we need to install it, following the instruction here for both the recommended font for the terminal of choice and p10k installation for Oh My Zsh.

After restart, type p10k configure if the configuration wizard doesn’t start automatically. Invoke the command whenever you want to re-configure.

Install tmux#

tmux is a terminal multiplexer. It lets you switch easily between several programs in one terminal, detach them (they keep running in the background) and reattach them to a different terminal.

brew install tmux

Vim#

Vim is a greatly improved version of the good old UNIX editor Vi.

Most likely, you have Vim pre-installed and would just need to add some configuration files to enrich the user experience.

Install VSC#

Visual Studio Code is a source-code editor made by Microsoft for Windows, Linux and macOS.

Download from the official website.
